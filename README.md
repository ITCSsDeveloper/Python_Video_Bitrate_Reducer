
 Video Bitrate Reducer - คู่มือการใช้งาน
========================================

ไฟล์ที่จำเป็นในโฟลเดอร์นี้:
---------------------------------
1. VideoConverter.exe  - โปรแกรมหลัก (คลิกเพื่อเปิดใช้งาน)
2. ffmpeg.exe          - ไลบรารีสำหรับแปลงวิดีโอ (จำเป็น)
3. ffprobe.exe         - ไลบรารีสำหรับอ่านข้อมูลวิดีโอ (จำเป็น)

วิธีใช้งาน:
---------------------------------
1. คลิกเปิด VideoConverter.exe
2. เลือก Input Folder (โฟลเดอร์ที่มีไฟล์วิดีโอที่ต้องการแปลง)
3. เลือก Output Folder (โฟลเดอร์สำหรับเก็บไฟล์ที่แปลงแล้ว)
   - ถ้าไม่เลือก จะสร้างโฟลเดอร์ "Output" ในโฟลเดอร์ Input อัตโนมัติ
4. ตั้งค่าเปอร์เซ็นต์การลด Bitrate (ค่าเริ่มต้น 30%)
5. ตั้งค่าจำนวนไฟล์ที่จะประมวลผลพร้อมกัน (ค่าเริ่มต้น 4 ไฟล์)
6. กดปุ่ม "เริ่มแปลง (Start Conversion)"
7. รอจนกว่าการแปลงจะเสร็จสมบูรณ์

หมายเหตุ:
---------------------------------
- โปรแกรมใช้ GPU Encoder (h264_amf) สำหรับ AMD GPU
- รองรับไฟล์วิดีโอ: .mp4, .mov, .mkv, .avi, .webm, .flv
- ไฟล์ทั้ง 3 ต้องอยู่ในโฟลเดอร์เดียวกันเพื่อให้โปรแกรมทำงานได้
- สามารถย้ายโฟลเดอร์นี้ไปไว้ที่ใดก็ได้

ปัญหาที่อาจพบ:
---------------------------------
Q: โปรแกรมบอกว่าไม่พบ FFmpeg
A: ตรวจสอบว่าไฟล์ ffmpeg.exe และ ffprobe.exe อยู่ในโฟลเดอร์เดียวกับ VideoConverter.exe

Q: เกิด Error: Unknown encoder 'h264_amf'
A: GPU ของคุณไม่รองรับ AMD encoder หรือไดรเวอร์ไม่ถูกต้อง
   ลองอัพเดทไดรเวอร์ GPU หรือติดต่อผู้พัฒนา

Q: การแปลงล้มเหลวบางไฟล์
A: ตรวจสอบว่าไฟล์วิดีโอไม่เสียหาย หรือลองลด bitrate น้อยลง
